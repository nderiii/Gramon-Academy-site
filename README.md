# Gramon Christian School Website

Welcome to the official repository for the Gramon Christian School website. This is a static, modern, and responsive website designed to showcase the school's mission, academic programs, and community.

## 🏫 About Gramon Christian School

Gramon Christian School is a CBC-based Christian institution located in Nairobi, Kenya. Since its founding in 2010, the school has been dedicated to providing a Christ-centered education that nurtures learners in character, faith, and excellence.

### 🌟 Our Core Pillars
- **Mission:** To provide a Christ-centered, CBC-aligned education that nurtures every child's potential and prepares them to thrive in life and eternity.
- **Vision:** To raise a generation of servant leaders who live with purpose, walk in faith, and impact the world.
- **Core Values:** Faith, Excellence, Integrity, and Compassion.

## 🚀 Features

- **Responsive Design:** Optimized for desktops, tablets, and mobile devices using Tailwind CSS.
- **CBC Integrated:** Highlights the implementation of Kenya's Competency-Based Curriculum.
- **Dynamic UI:** Smooth scrolling, interactive navigation, and mobile-friendly menus.
- **WhatsApp Integration:** Floating WhatsApp button for direct communication with the admissions office.
- **Comprehensive Sections:**
  - **Home:** Overview of the school and quick highlights.
  - **About Us:** Detailed history, mission, vision, and core values.
  - **Academic Program:** Information on CBC implementation and learning approach.
  - **Student Life:** Extracurricular activities and school environment.
  - **Admissions:** Information for prospective parents.
  - **Contact:** Location map and contact details.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure for accessibility and SEO.
- **Tailwind CSS:** Utility-first CSS framework for modern styling.
- **JavaScript (Vanilla):** Custom scripts for menu toggles and smooth scrolling.
- **Font Awesome:** Iconography for a professional look.
- **Google Fonts:** Premium typography for better readability.

## 💻 Local Development

To view the website locally:

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Alternatively, use a local server like **Live Server** (VS Code extension) for real-time updates.
